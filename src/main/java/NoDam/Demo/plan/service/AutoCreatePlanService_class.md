# Class 책임
- 날짜별 plan 자동 생성 서비스
- Trip isPlanning 상태를 관리하며 비동기(@Async)로 plan 생성
# 함수

특정 장소 추천 -> ai :: 추후 구현
일별 모든 장소 재 생성 -> ai :: autoGeneratePlans
모든 theme의 date plan 생성 -> ai :: autoGenerateDatePlans

모든 plan 생성 (처음 생성할 date plan 생성 이후 plan들 생성에 사용함) :: autoGenerateAllPlans
한개의 trip theme의 date plan 생성 : 사용하는 곳이 없음!

- public List<DatePlan> autoGenerateAllPlan(Trip trip, List<TripDate> datePlans) : 전체 날짜 plan 일괄 생성 (비동기)
- public DatePlan autoGenerateDatePlan(Trip trip, TripDate targetDate) : 단일 날짜 plan 생성 (비동기)
- public List<Place> autoRecommendPlace(Trip trip, TripDate targetDate, Plan beforePlan) : 장소 추천 (미구현)
